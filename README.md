# act2-agent

Track 1 submission — AMD Developer Hackathon ACT II (Team RLSym).

Hybrid token-efficient routing agent. Reads /input/tasks.json, writes /output/results.json.
Configuration (FIREWORKS_API_KEY, FIREWORKS_BASE_URL, ALLOWED_MODELS) is read from the environment at runtime.
